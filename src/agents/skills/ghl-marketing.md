## Tone & Voice Rules

- Direct, practical, slightly irreverent
- Speak like a GHL power user talking to another GHL user
- No corporate language, no buzzwords, no hype
- Short sentences. Short paragraphs. Get to the point.
- Never use em dashes
- Never use hashtags in body copy (Instagram only, in a separate hashtag block)
- Never say "game-changer", "revolutionary", "unlock your potential", or any generic SaaS marketing language
- Use specific GHL terminology: workflows, automations, pipelines, funnels, SaaS mode, merge fields, calendars, triggers, actions
- The Founders Offer is real and limited — treat it with genuine urgency, not manufactured hype

## Paid Ads Guidelines

### Meta Pixel Events (already installed)
- PageView: every page load
- Lead: user sign-up
- InitiateCheckout: upgrade click
- Purchase: subscription activation
- Pixel ID: 927054375981982

### UTM Parameters (mandatory on all paid links)
- utm_source=facebook / utm_source=instagram
- utm_medium=paid_social
- utm_campaign=founders_offer (for offer creative)
- utm_campaign=awareness (for awareness/video)
- utm_content=[ad_name]

### Ad Copy Structure
- **Hook:** 1 sentence. Hit a pain point or make a bold claim.
- **Problem:** 2-3 sentences. Describe the frustration in specific GHL terms.
- **Solution:** 2-3 sentences. Show what GHL Support Specialist does.
- **Proof:** 1 sentence. "Updated daily from GHL's live documentation."
- **CTA:** Founders Offer + code + URL.

### Retargeting Copy
- Acknowledge they visited but didn't sign up
- Restate the value prop concisely
- Re-emphasise the Founders Offer urgency

### KPI Thresholds (kill/scale rules)
- CTR below 0.8% after 500 impressions → pause ad set
- CPA above $25 → review creative
- CPA below $20 → scale ad set
- Target CPC: under $1.50
- Target sign-up to paid conversion: 20-30%

## Creative Direction

### Brand Colours
- Primary blue: #1a56db
- Background: dark navy or slate
- Text: white
- Avoid generic "AI blue" gradients

### Static Ad Format
- Dark background, large white headline text
- Product screenshot or chat UI mockup
- Feature tags as pills/badges
- CTA button in brand blue

### Video Ad Format (15-30 sec)
- Hook (0-3s): frustration moment — typing GHL question, waiting
- Problem (3-10s): support tickets, YouTube, Facebook groups
- Solution (10-20s): screen recording of product answering a real question
- CTA (20-30s): Founders Offer, code, link
- Subtitles on all video. No stock footage.
